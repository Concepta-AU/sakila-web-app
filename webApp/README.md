# Sakila Webapp

This is the frontend code for the Sakila test application. It is build using [Svelte](https://svelte.dev/).

## Developing

To install or update dependencies run:

```sh
npm install
```

(or shorter `npm i`)

To start a local development server for the frontend:

```sh
npm run dev
```

Or start the server and open the app in a new browser tab:

```sh
npm run dev -- --open
```

## Building

To create a production version of the frontend:

```sh
npm run build
```

You can preview the production build with `npm run preview`.
